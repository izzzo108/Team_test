# Проект: Pixel Team Game

Учебный сайт с пиксель-артовой мини-игрой. Цель проекта — не сама игра, а тренировка
совместной работы через Git/GitHub в команде. Подробности процесса — в README.md.

## Стек

- Чистый HTML/CSS/JS, без сборки и без фреймворков.
- Игра рендерится через `<canvas>` (`game.js`), стили — обычный CSS (`style.css`).
- Ничего не нужно устанавливать: `index.html` открывается прямо в браузере.

## Конвенции

- Один файл — одна логическая часть: `index.html` (разметка), `style.css` (стили),
  `game.js` (логика игры). Если игра разрастётся — разбивайте `game.js` на модули
  (`<script type="module">`), а не превращайте его в один большой файл.
- Комментарии — только там, где неочевидно "почему", а не "что".
- Коммиты и ветки — по правилам из `.claude/skills/git-workflow/SKILL.md` (GitHub Flow,
  Conventional Commits).

## Агенты и скиллы

В `.claude/agents/` и `.claude/skills/` лежит минимальный набор, адаптированный из
проекта [Everything Claude Code](https://github.com/affaan-m/everything-claude-code) (MIT).
Что для чего — см. README.md, раздел "Как пользоваться агентами".
